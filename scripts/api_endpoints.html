Create an address

POST 
/sse/config/v1/addresses

Create an address.

Request​

Query Parameters

    folder stringrequired

    Possible values: Value must match regular expression ^[0-9a-zA-Z._-\s]{1,}$, [Shared, Mobile Users, Remote Networks, Service Connections, Mobile Users Container, Mobile Users Explicit Proxy]

    The folder on which you want to perform this operation.

  • application/json

Body

The address you want to create

    oneOf
    • ip_netmask
    • ip_range
    • ip_wildcard
    • fqdn
    ip_netmask stringrequired
    ip_range stringrequired
    ip_wildcard stringrequired
    fqdn stringrequired

    Possible values: non-empty and <= 255 characters, Value must match regular expression ^[a-zA-Z0-9_]([a-zA-Z0-9._-])+[a-zA-Z0-9]$

    description string

    Possible values: <= 1023 characters

    name stringrequired

    Possible values: <= 63 characters

    Alphanumeric string [ 0-9a-zA-Z._-]

    tag string[]

    Possible values: <= 64

    Tags for address object
